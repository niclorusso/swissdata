# SwissData

An interactive web app for exploring real statistics about Switzerland's economy, demographics, and society. Built with data from official Swiss sources.

## Features

- **Dashboard** — Key indicators across economic, demographic, and social categories with trend charts
- **Canton Map** — Interactive SVG map of Switzerland's 26 cantons with tooltips and data overlays
- **Quiz** — Test your knowledge of Swiss statistics with 50+ questions and score tracking
- **Multilingual** — i18n support with language switching

## Data Sources

- [BFS](https://www.bfs.admin.ch) — Federal Statistical Office
- [SECO](https://www.seco.admin.ch) — State Secretariat for Economic Affairs
- [BAG](https://www.bag.admin.ch) — Federal Office of Public Health
- [OpenData.swiss](https://opendata.swiss) — Swiss Open Government Data
